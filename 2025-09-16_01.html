n8n 可整合的 60 個開源軟體

探索 n8n 自動化流程中可以連接的強大開源工具

1. Baserow

以 PostgreSQL 為基底的開源資料庫表格平台。

2. Bitwarden

密碼管理工具，核心功能以開源模式提供。

3. Cal.com

開源的行程預約日曆平台。

4. Cockpit

Red Hat 主導的 Linux 伺服器網頁管理工具，GPLv2 授權。

5. Cortex

TheHive 專案相關的開源事件分析與自動化平臺。

6. CrateDB

分散式 SQL 資料庫，具備開源社群版本 (Apache 2.0)。

7. crowd.dev

用於社群管理與洞察分析的開源平台。

8. DFIR-IRIS

又稱 IRIS，開源數位鑑識 / 威脅情報平台。

9. Discourse

由 Jeff Atwood 和 Robin Ward 創立的開源論壇 / 討論平臺 (GPLv2)。

10. ERPNext

一套涵蓋 ERP/CRM/HRM 等模組的開源系統 (GPLv3)。

11. Ghost

Node.js 上的開源部落格與內容管理平台 (MIT)。

12. GitLab (CE)

版本控管與 CI/CD 平台，社群版為開源 (MIT)。

13. Gotify

Self-hosted 推播服務，開源 (MIT)。

14. Grafana

可視化監控平台的核心部分開源 (AGPLv3)。

15. GraphQL

開放規範，其官方參考實作 (graphql-js 等) 為開源 (MIT)。

16. Grist

開源線上資料庫/協作表格 (Apache 2.0)。

17. Home Assistant

家居自動化平台，100% 開源 (Apache 2.0)。

18. Invoice Ninja

開源發票與帳單管理系統，有社群版 (GPLv3)。

19. Jenkins

著名的 CI/CD 自動化伺服器 (MIT)。

20. Kafka

Apache Software Foundation 所屬的分散式串流平台 (Apache 2.0)。

21. KoBoToolbox

用於線上/離線問卷、調查的開源工具集 (AGPLv3)。

22. Magento 2

Adobe 旗下的電商平台，有開源社群版本 (OSL 或 AFL)。

23. Matrix

即時通訊開放協定；主要實作 Synapse 等為開源 (Apache 2.0)。

24. Mattermost

企業級即時通訊協作平台，開源社群版 (MIT)。

25. Mautic

開源行銷自動化工具 (GPLv3)。

26. Metabase

商業智慧 (BI) 與資料視覺化開源工具 (AGPLv3)。

27. MISP

威脅情報分享與事件管理平臺 (GPLv3)。

28. Monica CRM

個人關係管理 (Personal CRM) 開源專案 (AGPLv3)。

29. MySQL

以 GPL 授權為主的知名開源關聯式資料庫。

30. Nextcloud

自架式雲端檔案同步與協作平台 (AGPLv3)。

31. NocoDB

將任何 SQL 資料庫轉換成類 Airtable 介面之開源平台 (GPLv3)。

32. npm

Node.js 的套件管理器，CLI 等部分在 GitHub 上開源。

33. Odoo (CE)

ERP/CRM/電商一體化套件，開源社群版 (LGPLv3)。

34. OpenCTI

開源的網路威脅情報平台 (GNU Affero GPLv3)。

35. OpenThesaurus

一個開源的同義詞詞典專案 (AGPL 或類似開源授權)。

36. Postgres

老牌開源關聯式資料庫 (PostgreSQL License)。

37. PostHog

開源產品分析 (Product Analytics) 平臺 (AGPLv3)。

38. QuestDB

高性能時間序列資料庫，開源社群版 (Apache 2.0)。

39. QuickChart

提供圖表生成的 API 工具，本身有開源後端 (MIT)。

40. RabbitMQ

基於 Erlang/OTP 的 AMQP 佇列系統 (MPL)。

41. Redis

高速鍵值儲存資料庫 (BSD 3-Clause)。

42. Rocket.Chat

即時通訊與協作平台 (MIT)。

43. Rundeck

運維編排與工作流自動化工具 (Apache 2.0)。

44. SeaTable

以表格/看板介面為主的協作平台，有社群開源版。

45. Sentry.io

程式錯誤追蹤與監控平台，核心部分為開源 (BSD 2-Clause)。

46. Stable Diffusion

深度學習生成式模型，由 Stability AI 等社群共同維護。

47. Strapi

Node.js 上的開源 Headless CMS (MIT)。

48. Supabase

封裝 PostgreSQL 的後端即服務 (Apache 2.0)。

49. Sysdig

雲端原生可觀測性與安全工具，核心 sysdig 工具為開源 (GPLv2)。

50. Taiga

開源專案管理與敏捷開發平臺 (AGPLv3)。

51. TheHive

開源安全事件響應與威脅情報平台 (AGPLv3)。

52. TimescaleDB

以 PostgreSQL 為基礎的時間序列擴充，有社群開源部分。

53. Twake

開源團隊協作平台 (AGPLv3)。

54. Wekan

類似 Trello 的看板式專案管理工具，開源 (MIT)。

55. WooCommerce

WordPress 的開源電商外掛 (GPLv3)。

56. WordPress

最廣泛使用的開源部落格/網站平台 (GPLv2)。

57. YOURLS

開源短網址服務 (MIT)。

58. Zabbix

開源監控軟體 (GPLv2)。

59. Zammad

開源客服與工單管理系統 (GPLv3)。

60. Zulip

開源即時通訊平台 (Apache 2.0)。
